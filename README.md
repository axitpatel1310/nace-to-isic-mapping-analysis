# NACE to ISIC Mapping Analysis

This project maps 25 NACE Rev.2 classes to their closest ISIC Rev.4 equivalents.  
The goal is to understand how international industry classification systems align and where they differ.

---

## 📌 Objective

- Select 25 NACE Rev.2 classes  
- Map each to ISIC Rev.4  
- Classify mapping type:
  - One-to-One  
  - One-to-Many  
  - Approximate  
- Provide short rationale for each  

---

## ⚙️ Methodology

1. Used official NACE and ISIC documentation  
2. Compared class definitions (not just codes)  
3. Identified closest ISIC match  
4. Determined match type based on scope:
   - Same scope → One-to-One  
   - NACE broader → One-to-Many  
   - No exact match → Approximate  

---

## 📊 Visualizations

The project includes:
- Bar chart → distribution of match types  
- Pie chart → percentage breakdown  

These help highlight how often classifications align exactly versus where structural differences exist.
